## About:

Opening Soon lists movies new movies in theaters soon listed on IMDb.com and
gives info on each movie.

## Installation:

Add this line to your application's Gemfile:

```ruby
gem 'coming_soon_refactored'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem coming_soon_refactored
